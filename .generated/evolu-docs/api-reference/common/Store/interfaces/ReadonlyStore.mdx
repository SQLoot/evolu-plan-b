A read-only view of a [Store](/docs/api-reference/common/Store/interfaces/Store.mdx) that provides state access and change
notifications without allowing modifications.

Use [ReadonlyStore](/docs/api-reference/common/Store/interfaces/ReadonlyStore.mdx) in public APIs where consumers should observe state
but not modify it directly.
