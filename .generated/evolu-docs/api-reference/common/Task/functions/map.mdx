Maps values to [Task](/docs/api-reference/common/Task/type-aliases/Task.mdx)s, failing fast on first error.

Sequential by default — use [parallel](/docs/api-reference/common/Task/functions/parallel.mdx) for parallel execution.

### Example

```ts
const result = await run(map(userIds, fetchUser));
if (!result.ok) return result;
// result.value: ReadonlyArray<User>
```
