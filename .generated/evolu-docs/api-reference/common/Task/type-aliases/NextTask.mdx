A [Task](/docs/api-reference/common/Task/type-aliases/Task.mdx) that can complete with a value, signal done, or fail.

Forms a parallel with [NextResult](/docs/api-reference/common/Result/type-aliases/NextResult.mdx):

- `Result<A, E>` → `NextResult<A, E, D>`
- `Task<T, E>` → `NextTask<T, E, D>`

Use for pull-based protocols like iterators where `Done<D>` signals normal
completion rather than an error.

| `E` | `never` |
