An [Owner](/docs/api-reference/common/local-first/Owner/interfaces/Owner.mdx) for sharding data.

ShardOwners are the recommended storage location for most application data
because they can be completely deleted (both on relays and devices) and
conditionally synced.

Can be created from [OwnerSecret](/docs/api-reference/common/local-first/Owner/variables/OwnerSecret.mdx) via [createShardOwner](/docs/api-reference/common/local-first/Owner/functions/createShardOwner.mdx) or
deterministically derived from [AppOwner](/docs/api-reference/common/local-first/Owner/interfaces/AppOwner.mdx) using
[deriveShardOwner](/docs/api-reference/common/local-first/Owner/functions/deriveShardOwner.mdx).
