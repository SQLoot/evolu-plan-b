Creates a new random [OwnerWriteKey](/docs/api-reference/common/local-first/Owner/variables/OwnerWriteKey.mdx) for rotation.

The initial OwnerWriteKey is deterministically derived from
[OwnerSecret](/docs/api-reference/common/local-first/Owner/variables/OwnerSecret.mdx). Use `createOwnerWriteKey` to rotate (replace) the write
key without changing the owner identity.
