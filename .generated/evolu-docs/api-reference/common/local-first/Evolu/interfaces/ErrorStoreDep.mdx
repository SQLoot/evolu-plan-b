Defined in: [packages/common/src/local-first/Evolu.ts:686](https://github.com/SQLoot/EvoLoot/blob/d86280a21ced8e9f86b70217b1c345b37bea4ce2/packages/common/src/local-first/Evolu.ts#L686)

Shared error store for all Evolu instances. Subscribe once to handle errors
globally across all instances.

### Example

```ts
deps.evoluError.subscribe(() => {
  const error = deps.evoluError.get();
  if (!error) return;
  console.error(error);
});
```
