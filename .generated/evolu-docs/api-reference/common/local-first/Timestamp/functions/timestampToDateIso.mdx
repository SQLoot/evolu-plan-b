Convert a [Timestamp](/docs/api-reference/common/local-first/Timestamp/variables/Timestamp.mdx) to an ISO 8601 [DateIso](/docs/api-reference/common/Type/variables/DateIso.mdx) string.

The conversion uses the timestamp's `millis` (a [Millis](/docs/api-reference/common/Time/variables/Millis.mdx) value) and
`Date.prototype.toISOString()` to produce a `DateIso`.
