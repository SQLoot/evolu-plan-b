Defined in: [packages/common/src/local-first/Kysely.ts:182](https://github.com/SQLoot/EvoLoot/blob/d86280a21ced8e9f86b70217b1c345b37bea4ce2/packages/common/src/local-first/Kysely.ts#L182)

The SQLite `json_object` function.

### Example

```ts
import { kysely } from "@evolu/common";

// TODO: Update for Evolu
const result = await db
  .selectFrom("person")
  .select((eb) => [
    "id",
    kysely
      .jsonBuildObject({
        first: eb.ref("first_name"),
        last: eb.ref("last_name"),
        full: kysely.sql<string>`first_name || ' ' || last_name`,
      })
      .as("name"),
  ])
  .execute();

result[0]?.id;
result[0]?.name.first;
result[0]?.name.last;
result[0]?.name.full;
```

The generated SQL (SQLite):

```sql
select "id", json_object(
  'first', first_name,
  'last', last_name,
  'full', "first_name" || ' ' || "last_name"
) as "name"
from "person"
```
