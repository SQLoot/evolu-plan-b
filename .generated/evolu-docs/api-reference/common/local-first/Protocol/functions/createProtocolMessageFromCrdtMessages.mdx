Creates a [ProtocolMessage](/docs/api-reference/common/local-first/Protocol/type-aliases/ProtocolMessage.mdx) from CRDT messages.

If the message size would exceed [defaultProtocolMessageMaxSize](/docs/api-reference/common/local-first/Protocol/variables/defaultProtocolMessageMaxSize.mdx), the
protocol ensures all messages will be sent in the next round(s) even over
unidirectional and stateless transports.
