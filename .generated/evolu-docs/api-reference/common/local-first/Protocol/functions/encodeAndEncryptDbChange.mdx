Encodes and encrypts a [DbChange](/docs/api-reference/common/local-first/Storage/variables/DbChange.mdx) using the provided owner's encryption
key. Returns an encrypted binary representation as [EncryptedDbChange](/docs/api-reference/common/local-first/Storage/type-aliases/EncryptedDbChange.mdx).

The format includes the protocol version for backward compatibility and the
timestamp for tamper-proof verification that the timestamp matches the change
data.
