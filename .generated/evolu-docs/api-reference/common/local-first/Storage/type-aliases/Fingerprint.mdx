A cryptographic hash used for efficiently comparing collections of
[TimestampBytes](/docs/api-reference/common/local-first/Timestamp/variables/TimestampBytes.mdx)es.

It consists of the first [fingerprintSize](/docs/api-reference/common/local-first/Storage/variables/fingerprintSize.mdx) bytes of the SHA-256 hash of
one or more timestamps.
