An order for numbers in ascending order.

### Example

```ts
orderNumber(1, 2); // -1
orderNumber(2, 1); // 1
orderNumber(1, 1); // 0
[2, 1, 3].toSorted(orderNumber); // [1, 2, 3]
reverseOrder(orderNumber)(1, 2); // 1
reverseOrder(orderNumber)(2, 1); // -1
reverseOrder(orderNumber)(1, 1); // 0
```
