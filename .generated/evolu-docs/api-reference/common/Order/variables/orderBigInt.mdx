An order for bigints in ascending order.

### Example

```ts
orderBigInt(1n, 2n); // -1
orderBigInt(2n, 1n); // 1
orderBigInt(1n, 1n); // 0
[2n, 1n, 3n].toSorted(orderBigInt); // [1n, 2n, 3n]
```
