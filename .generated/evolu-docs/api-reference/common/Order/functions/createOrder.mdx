Creates an ordering function from a "less than" comparator.

### Example

```ts
const orderNumber = createOrder<number>((x, y) => x < y);
expect(orderNumber(1, 2)).toEqual(-1);
expect(orderNumber(2, 1)).toEqual(1);
expect(orderNumber(1, 1)).toEqual(0);
```
