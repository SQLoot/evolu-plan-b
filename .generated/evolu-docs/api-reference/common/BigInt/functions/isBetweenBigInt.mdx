Creates a predicate that checks if a BigInt is within a range, inclusive.

### Example

```ts
const isBetween10And20 = isBetweenBigInt(10n, 20n);
console.log(isBetween10And20(15n)); // true
console.log(isBetween10And20(25n)); // false
```
