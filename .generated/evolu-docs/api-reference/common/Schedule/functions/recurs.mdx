A schedule that recurs a fixed number of times.

Convenience for `take(n)(forever)`. Outputs the current repetition count (0,
1, 2, ..., n-1).

### Example

```ts
// Retry up to 3 times (4 total attempts including initial)
const retry = recurs(3);
```
