Fibonacci backoff schedule.

Delays follow the Fibonacci sequence, growing more slowly than exponential:

- Attempt 1: `initial`
- Attempt 2: `initial`
- Attempt 3: `initial * 2`
- Attempt 4: `initial * 3`
- Attempt 5: `initial * 5`
- ...

Never stops — combine with [take](/docs/api-reference/common/Schedule/functions/take.mdx) or [maxElapsed](/docs/api-reference/common/Schedule/functions/maxElapsed.mdx) to limit.

### Example

```ts
// 100ms, 100ms, 200ms, 300ms, 500ms, 800ms, ...
const fib = fibonacci("100ms");
```
