Defined in: [packages/common/src/Schedule.ts:969](https://github.com/SQLoot/EvoLoot/blob/d86280a21ced8e9f86b70217b1c345b37bea4ce2/packages/common/src/Schedule.ts#L969)

Transforms the output of a schedule.

The delay (second tuple element) remains unchanged.

### Example

```ts
import { exponential, mapSchedule } from "@evolu/common";

const schedule = mapSchedule((delay) => ({
  delay,
  doubled: delay * 2,
}))(exponential("100ms"));
```
