Collects all outputs into an array.

Each step outputs an array containing all outputs so far.

### Example

```ts
// Collect all delays
const collected = collectAllScheduleOutputs(take(3)(spaced("100ms")));
// Outputs: [100], [100, 100], [100, 100, 100]
```
