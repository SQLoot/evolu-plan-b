A schedule that runs through a sequence of delays.

Convenience for sequencing single-delay schedules. Useful for predefined
retry patterns.

### Example

```ts
// Custom retry sequence: 100ms, 500ms, 2s
const custom = fromDelays("100ms", "500ms", "2s");
```
