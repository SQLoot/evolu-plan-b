Defined in: [packages/common/src/Schedule.ts:1196](https://github.com/SQLoot/EvoLoot/blob/d86280a21ced8e9f86b70217b1c345b37bea4ce2/packages/common/src/Schedule.ts#L1196)

Collects outputs while a predicate is true.

More flexible than [collectAllScheduleOutputs](/docs/api-reference/common/Schedule/functions/collectAllScheduleOutputs.mdx) — stops collecting when
the predicate returns false.

### Example

```ts
// Collect delays while under 1 second
const smallDelays = collectWhileScheduleOutput(
  (delay: Millis) => delay < 1000,
)(exponential("100ms"));
// Outputs: [100], [100, 200], [100, 200, 400], [100, 200, 400, 800], stops
```
