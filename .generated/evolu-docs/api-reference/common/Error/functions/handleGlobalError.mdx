Forwards an error to a [GlobalErrorScope](/docs/api-reference/common/Error/interfaces/GlobalErrorScope.mdx)'s `onError` callback.

Asserts that `onError` is set, then normalizes the error with
[createUnknownError](/docs/api-reference/common/Error/functions/createUnknownError.mdx) and calls the callback.
