Creates a [Ref](/docs/api-reference/common/Ref/interfaces/Ref.mdx) with the given initial state.

By default, state is always updated. We can provide an optional [Eq](/docs/api-reference/common/Eq/type-aliases/Eq.mdx)
function as the second argument to skip updates when the new state equals the
current state.
