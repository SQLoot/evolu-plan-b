Creates a [ConsoleOutput](/docs/api-reference/common/Console/interfaces/ConsoleOutput.mdx) that writes to `globalThis.console`.

Pure transport - just calls the native console method with the entry args.
Use [createConsoleEntryFormatter](/docs/api-reference/common/Console/functions/createConsoleEntryFormatter.mdx) with [ConsoleConfig.formatEntry](/docs/api-reference/common/Console/interfaces/ConsoleConfig.mdx#formatentry)
for timestamps and path prefixes.

### Example

```ts
const output = createNativeConsoleOutput();
```
