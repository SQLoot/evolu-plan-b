Concatenates two arrays, returning a new readonly array.

Returns a non-empty array when at least one input is non-empty.

### Example

```ts
concatArrays([1, 2], [3, 4]); // [1, 2, 3, 4]
concatArrays([], [1]); // [1] (non-empty)
concatArrays([1], []); // [1] (non-empty)
```
