Returns a new readonly array with elements removed and/or replaced.

Wraps native `toSpliced`.

### Example

```ts
spliceArray([1, 2, 3, 4], 1, 2); // [1, 4]
spliceArray([1, 2, 3], 1, 1, 10, 11); // [1, 10, 11, 3]
```
