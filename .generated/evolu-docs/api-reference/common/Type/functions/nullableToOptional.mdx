Converts each “nullable” property (a union that includes [Null](/docs/api-reference/common/Type/variables/Null.mdx)) into an
[optional](/docs/api-reference/common/Type/functions/optional.mdx) property. This means consumers can omit the property
entirely, or set it to `null`, or set it to the non-null member of the
union.
