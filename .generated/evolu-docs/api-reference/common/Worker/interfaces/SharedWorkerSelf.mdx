Typed `self` for code running inside a shared worker.

This is the worker-side counterpart to [SharedWorker](/docs/api-reference/common/Worker/interfaces/SharedWorker.mdx). It wraps `self`
inside the shared worker, providing typed `onConnect` callbacks.
