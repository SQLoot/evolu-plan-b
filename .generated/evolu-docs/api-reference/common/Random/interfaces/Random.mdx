A simple wrapper around Math.random().

For more complex needs check [RandomLibDep](/docs/api-reference/common/Random/interfaces/RandomLibDep.mdx).

### Example

```ts
// For apps
const random = createRandom();
random.next();

// For tests
const random = createRandomWithSeed("test");
random.next();
```
