A promise that resolves when a termination signal is received.

Resolves on `SIGINT` (Ctrl-C), `SIGTERM` (OS/k8s/Docker termination),
`SIGHUP` (console close/terminal disconnect), or `SIGBREAK` (Windows
Ctrl-Break).
